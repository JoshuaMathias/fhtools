del
functions in module plumbing

Description

Low-level call to delete to a specific REST endpoint from FamilySearch

Usage

del(url[, opts][, headers][, responseMapper]);

Parameters

ParamTypeDetails
urlString

may be relative

opts
(optional)
Object

options to pass to the http function specified during init

headers
(optional)
Object

options headers

responseMapper
(optional)
Function

function to map response data to something else

Returns

Object

a promise that behaves like promises returned by the http function specified during init
