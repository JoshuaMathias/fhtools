getSpouses
functions in module person

Description

Get the relationships to a person's spouses. The response includes the following convenience functions

  • getCoupleRelationships() - an array of Couple relationships
  • getChildAndParentsRelationships() - an array of ChildAndParents relationships for children of the couples
  • getPerson(pid) - a Person for any person id in a relationship except children
