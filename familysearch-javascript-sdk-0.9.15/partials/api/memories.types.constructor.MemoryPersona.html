MemoryPersona
type in module memories

Description

Memory Persona (not a true persona; can only contain a name and a media artifact reference)

FamilySearch API Docs

Usage

MemoryPersona([data]);

Parameters

ParamTypeDetails
data
(optional)
Object

an object with optional attributes {$memoryId, name, memoryArtifactRef}. To create a new memory persona, you must set $memoryId and name. name can be a Name object or a fullText string. NOTE memory persona names don't have given or surname parts, only fullText
