getAccessTokenForMobile
functions in module authentication

Description

Get the access token for the user, passing in their user name and password Call this only for mobile apps; otherwise call getAccessToken

You don't need to store the access token returned by this function; you just need to ensure that the promise returned by this function resolves before making calls that require authentication.
