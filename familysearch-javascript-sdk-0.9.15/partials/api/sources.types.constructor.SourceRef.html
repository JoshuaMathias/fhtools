SourceRef
type in module sources

Description

Reference from a person or relationship to a source. To create a new SourceRef you must set sourceDescription and either $personId, $coupleId, or $childAndParentsId

FamilySearch API Docs: Person SourceRef, Couple SourceRef, and ChildAndParents SourceRef

Usage

SourceRef([data]);

Parameters

ParamTypeDetails
data
(optional)
Object

an object with optional attributes {$personId, $coupleId, $childAndParentsId, sourceDescription, tags}. sourceDescription can be a SourceDescription, a source description id, or a source description URL. tags is an array (string[]) of tag names
