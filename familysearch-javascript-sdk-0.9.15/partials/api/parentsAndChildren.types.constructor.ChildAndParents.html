ChildAndParents
type in module parentsAndChildren

Description

Child and parents relationship

FamilySearch API Docs

Two methods to note below are $save and $delete. $save persists the changes made to father, mother, child, and facts; $delete removes the relationship.

Usage

ChildAndParents([data]);

Parameters

ParamTypeDetails
data
(optional)
Object

an object with optional attributes {father, mother, child, fatherFacts, motherFacts}. father, mother, and child are Person objects, URLs, or ids. fatherFacts and motherFacts are arrays of Facts or objects to be passed into the Fact constructor.
